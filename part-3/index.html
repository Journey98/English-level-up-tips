# Loved 爱过

我牵过许多女孩的手

也搂过许多女孩的腰

我看过许多云彩

躺在不同的女孩腿上睡着

我去过许多城市

在不同的女孩身边醒来

你要问我

分手了那么久 为什么还不忘掉？

忘掉

是薄情

不忘

是花心

我宁愿做个花心的大萝卜
